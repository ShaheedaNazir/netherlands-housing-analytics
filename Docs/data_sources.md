# Data Sources

This project uses Dutch open data sources.

## Planned Sources

### CBS Open Data

CBS Open Data provides official Dutch statistics from Statistics Netherlands.

Website:

https://opendata.cbs.nl

Planned datasets:

- Population by municipality
- Housing stock by municipality
- Migration by municipality
- Income or household statistics

### Amsterdam Open Data

Amsterdam Open Data provides datasets from the Municipality of Amsterdam.

Website:

https://data.amsterdam.nl

Planned datasets:

- Neighborhoods
- Districts
- Buildings
- Public space or housing-related data

## Notes

The first phase of this project starts with CBS population data.
