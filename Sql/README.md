# Netherlands Housing Analytics

This is a SQL Server portfolio project using Dutch open data.

The goal of this project is to analyze population, housing, and regional development trends in the Netherlands using public datasets from CBS and Dutch government data sources.

## Project Goals

- Practice SQL Server with real Dutch open data
- Build a professional GitHub portfolio project
- Learn data cleaning, data modeling, and analytical SQL
- Prepare for data analyst, BI analyst, and SQL developer roles in the Netherlands

## Tools Used

- SQL Server Management Studio 22
- Microsoft SQL Server
- Dutch open data
- GitHub
